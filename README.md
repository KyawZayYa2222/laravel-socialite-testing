### Installation
___
- clone repo
- composer install or composer update
- cp .env.example .env
- setup your credentials (.env)
  > #### Google Login
  > - GOOGLE_CLIENT_ID = 
  >  - GOOGLE_CLIENT_SECRET = 
  >  - GOOGLE_REDIRECT_URL = 
- php artisan migrate
- php artisan serve
- npm install
- npm run dev
